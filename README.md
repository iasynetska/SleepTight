# SleepTight

Mattress online store

## Getting Started

Clone the application and run a following commands in the root directory

```
npm install
```
```
npm run start
```

## Technologies

* HTML
* CSS
* [React.js](https://reactjs.org/)
* [Node.js](https://nodejs.org/en/)
* [Webpack](https://webpack.js.org/)
* [Babel](https://babeljs.io/)
